Smart GECI – Overview

Smart GECI (Smart Government Engineering College Idukki) is an intelligent campus-level geospatial decision support system designed to integrate Geographic Information Systems (GIS), spatial databases, and Artificial Intelligence for sustainable campus management.

The system transforms the physical campus into a digital twin by integrating spatial data, infrastructure data, environmental analytics, and real-time decision support mechanisms. It enables natural language interaction with campus data using Large Language Models (LLMs).

Objective of Smart GECI

The primary objective of Smart GECI is to:

Digitize campus infrastructure using GIS

Enable geospatial analysis using PostGIS

Provide conversational access to campus data

Support environmental sustainability planning

Demonstrate integration of AI and GIS in a smart campus setting

The system is designed as a proof-of-concept for applying intelligent geospatial technologies in educational institutions.

Core Technologies Used

Smart GECI integrates multiple modern technologies:

PostgreSQL with PostGIS for spatial database management

QGIS for spatial data preparation and visualization

Node.js backend for orchestration and API handling

Python NLP service for AI-based reasoning

Large Language Models (LLMs) for natural language understanding

Retrieval-Augmented Generation (RAG) for knowledge-based responses

Deterministic SQL generation for structured data queries

This hybrid architecture ensures both accuracy and intelligence.

Hybrid AI Architecture

Smart GECI uses a hybrid conversational architecture:

Structured Query Path (NL2SQL)

Used for numeric and database-driven queries

Generates deterministic SQL

Executes queries on PostgreSQL/PostGIS

Returns accurate, real-time data

Knowledge Retrieval Path (RAG)

Used for descriptive and conceptual questions

Retrieves relevant knowledge documents

Uses LLM to generate contextual answers

Geo Decision Support Path

Uses predefined spatial SQL templates

Supports nearest building queries

Supports spatial distance analysis

Supports buffer and proximity operations

This separation ensures reliability and flexibility.

Geospatial Capabilities

Smart GECI integrates spatial analysis capabilities such as:

Distance calculations between buildings

Proximity analysis

Spatial filtering

Buffer zone analysis

Geo-based decision support

Using PostGIS functions like ST_Distance and ST_DWithin, the system can answer spatial queries such as:

Which building is closest to the library?

Which buildings are within 100 meters of the admin block?

Identify zones with higher environmental impact.

Environmental Sustainability Integration

Smart GECI supports sustainable environmental management through:

Urban Heat Island (UHI) analysis

Land use and land cover mapping

Environmental data integration

Spatial visualization of heat zones

Support for tree plantation planning

The system demonstrates how GIS and machine learning can be combined for sustainable campus planning.

Natural Language Interface

One of the key features of Smart GECI is its conversational interface.

Users can interact using natural language queries such as:

How many floors are in the CSE building?

What is Smart GECI?

Which building is closest to the library?

The system classifies user intent and routes the query to:

SQL engine (for structured data)

RAG knowledge base (for explanations)

Spatial decision engine (for geo queries)

This makes Smart GECI a Conversational Geo-Intelligent Decision Support System.

Digital Twin Concept

Smart GECI acts as a digital twin of the campus by:

Representing buildings spatially

Storing infrastructure metadata

Integrating environmental analytics

Allowing intelligent querying

The digital twin enables administrators and planners to simulate and analyze campus-level decisions.

Research and Academic Significance

Smart GECI demonstrates:

Integration of GIS with Artificial Intelligence

Hybrid NL2SQL and RAG architecture

Practical implementation of spatial decision support

Conversational AI for structured databases

Data-aware intelligent systems

The project highlights how modern AI technologies can enhance geospatial systems and improve decision-making in smart campus environments.

System Limitations

Smart GECI is a prototype system designed for academic demonstration. While it supports intelligent query routing and spatial analysis, it operates on a limited dataset and is optimized for demonstration purposes rather than large-scale deployment.

Future improvements may include:

Real-time IoT integration

Larger-scale spatial analytics

Advanced predictive modeling

Automated spatial planning recommendations

Conclusion

Smart GECI represents an interdisciplinary integration of GIS, AI, spatial databases, and conversational intelligence. It demonstrates how hybrid AI architectures can enable intelligent geospatial decision support in educational institutions.

The system showcases how structured data querying, knowledge retrieval, and spatial reasoning can be unified into a single conversational platform.
